Literature Survey

Machine learning and image processing have become increasingly important in areas like skincare product recommendation, skin feature analysis, and user feedback analysis. Identifying skin features and issues is crucial for recommending a product, and a skincare product recommendation framework is proposed in [1], which includes skin type identification and acne detection. A system for detecting facial imperfections is proposed in [2], focusing on issues like acne, freckles, wrinkles, and moles. Skin color detection uses HSV and YCbCr color spaces for facial tone. Identifying popular skincare products in the market can be valuable for providing customized suggestions. This can be achieved using natural language processing and machine learning techniques to classify products based on product features and user reviews. Companies like Sephora use a combination of sales data and customer reviews to identify popular products and make recommendations [3]. Learning customer preferences on recommended skincare products is another important component of a skincare product recommendation system. A sentiment analysis and text classification model for social media content can be used to categorize sentiment in social media content [4]. User images can be analyzed using image processing to check for allergies caused by suggested products. Recommender systems [5] for fashion products and books are tested, with collaborative filtering being highlighted for harnessing user preferences. A food recommendation system is proposed in [6], using deep learning for product classification and genetic algorithms for filtering and recommending products based on preferences.

Research Problem & Solution

Research Problem

Skin health is crucial for physical and mental well-being. Common skin issues include oily or dry skin, pressure ulcers, and acne. Factors like genetics, environment, nutrition, stress, and sleep contribute to skin problems. Facial skin is particularly important for self-esteem and confidence. Acne, dark spots, wrinkles, and oily skin can cause mental stress, leading people to use cosmetics to conceal problems. However, these methods do not address the underlying issues. Factors like poverty, geography, and gender may discourage people from seeking dermatologists, leading to home remedies or over-the-counter medicines. As the skincare market advances, selecting the perfect product becomes more challenging.

Solution

This study introduces a skincare system that addresses common skin disorders in Sri Lanka, such as skin type, texture, acne, and dark spots, using an internet connection and camera. The system identifies these issues using computer vision and machine learning and offers suitable products. It also includes a product popularity recognition component, suggesting post-recommendation care, and analyzing user feedback using machine learning. It also collects user images and responses to identify allergies or complications. The system uses various tools, algorithms, and techniques to provide personalized recommendations based on user preferences and product popularity, enhancing the user experience. This system has the potential to revolutionize the skincare industry by making it easier for consumers to find and use products that suit their needs.

Given below are the research objectives we hoped to achieve with this research

Identify the features of the facial skin like type, texture and concerns like acne.

Sub Objectives

  1. Develop Machine Learning models to identify skin type.
  2. Develop machine learning models to determine the severity of acne.
  3. Implement image segmentation methods for best accuracy.
  4. Identify Regions of Interest (ROI) in images.
  5. Identify Regions of Interest (ROI) in images.

Analyze current skincare products in the market, determine their popularity among users

Sub Objectives

  1. Identify the most popular skincare products based on user reviews and ratings.
  2. Analyze the sentiment of user reviews for different skincare products.
  3. Evaluate the accuracy and reliability of the developed models.

Recommend skincare products based on identified skin features.

  1. Collect skin care product data (category, brand, price, popularity, ingredients etc.) from various sources.

  3. Implement image segmentation methods for best accuracy.

  2. Obtain user images after a certain time period to check for allergies due to recommended products and use computer vision techniques to analyze them.

Methodology

The complete proposed system for this project has four main components namely,
  1. 1. Identification of skin type, texture and the skin concerns acne and dark spots
  2. 2. Recommendation of skincare products based on the skin features identified.
  3. 3. Popularity analysis of skincare products for product recommendation
  4. 4. Feedback analysis and allergy detection for post recommendation care.

A user will upload a facial image to the first component, which will analyze and extract skin traits and problems. This step involves image processing methods that allow the system to apply machine learning algorithms to images and acquire accurate results. After obtaining a preprocessed image, it will be delivered to a deep learning model for feature categorization and extraction. The model will use a dataset of images with skin features and issues for training the model. Once the model has extracted all the necessary features from the image uploaded by the user, the system will display the diagnosis to the user and pass the information over to the product recommendation component. The final system will use the extracted skin feature information for recommending a personalized skincare treatment plan for the user. In order to do this the component will also use the products extracted from the product popularity analysis component. This component will search the internet for popular products in the market and analysis these products’ rating and user satisfaction. Then a selected list of products will be transferred over to the product recommendation component to be used to recommend to the users. Once a product recommendation is made by the component the user’s satisfaction and the effectiveness of the recommendation treatment plan need to be analyzed. For this the user feedback and allergy detection component is used. This component analyses the sentiment of the users’ feedback and uses it to improve the system. Further, it provides post recommendation care to the users by having the users upload image after using the recommended product to identify if any allergies or interaction might have occurred.
